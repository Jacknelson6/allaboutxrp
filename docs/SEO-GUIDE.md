# SEO Optimizer Guide

## Keyword Research & Strategy
- Focus on search intent (informational, navigational, transactional, commercial)
- Balance search volume with competition
- Target long-tail keywords for quick wins
- 1-2% keyword density, natural placement
- Primary keyword in: Title tag, H1, first paragraph, URL, meta description
- Use semantic variations and related terms

## On-Page SEO
- Title tags: under 60 chars, keyword near beginning, compelling
- Meta descriptions: 150-160 chars, includes keywords, call-to-action
- Proper H1 > H2 > H3 hierarchy
- Clean URL structure (/learn/what-is-xrp)
- Image optimization: descriptive alt text, WebP, lazy loading, dimensions specified

## E-E-A-T Principles
- Include author byline on every page (AllAboutXRP Editorial Team)
- Cite authoritative sources
- Show real experience and original insights
- Keep content accurate with dates
- Author bios and credentials

## Content Structure (every page)
1. H1 with primary keyword
2. Brief intro with keyword in first 100 words
3. "What is [Topic]?" section
4. "Why [Topic] Matters" section  
5. "How to" / practical guide section
6. Best Practices / Advanced tips
7. Common Mistakes to Avoid
8. FAQ section for featured snippets
9. Conclusion with CTA

## Content Length
- Blog posts: 1,500-2,500 words
- Product pages: 300-500 words minimum
- Category pages: 500-1,000 words

## Technical SEO
- Full Article schema: datePublished, dateModified, author (Person), publisher (Organization w/ logo)
- BreadcrumbList schema on all pages
- FAQ schema where applicable
- Canonical tags on every page
- Open Graph + Twitter Card meta tags
- robots.txt: disallow /api/ routes
- Sitemap with lastmod, changefreq, priority per URL

## Core Web Vitals
- LCP < 2.5s (preload fonts, CDN, lazy load images)
- FID < 100ms (defer non-critical JS, break up long tasks)
- CLS < 0.1 (set dimensions on images/video, reserve ad space)
- Preload critical fonts, async/defer scripts

## Mobile SEO
- Responsive mobile-first design
- 48x48px minimum touch targets
- 16px minimum font size
- Proper viewport config
- Fast mobile page speed
